# SIM-PRO: Sistema Industrial de Modelado y Pronóstico Estocástico

## 🚀 Descripción
SIM-PRO es una herramienta de ingeniería de software diseñada para la validación de procesos estocásticos mediante el algoritmo de Generador Congruencial Lineal (LCG) y la prueba de bondad de ajuste Ji-Cuadrada ($\chi^2$). 

Este proyecto fue desarrollado bajo los lineamientos del estándar **MoProSoft** para garantizar la calidad y trazabilidad en entornos de simulación industrial.

## 🛠️ Tecnologías Utilizadas
- **Backend:** PHP 8.2 (Programación Orientada a Objetos)
- **Frontend:** HTML5, CSS3 (Bootstrap 5.3) y JavaScript
- **Gráficos:** Chart.js API
- **Gestión:** Jira Software & GitHub
- **Servidor:** Apache 2.4 (XAMPP Environment)

## 📊 Validación Estadística
El sistema evalúa la uniformidad de los datos generados comparando el estadístico calculado contra un valor crítico de **9.488** (con 4 grados de libertad y un 95% de confianza).

## 📥 Instalación
1. Clonar el repositorio en tu carpeta local `htdocs`.
2. Asegurarse de tener PHP 8.2 configurado en XAMPP/WAMP.
3. Abrir el navegador en `http://localhost/sim-pro/index.php`.
